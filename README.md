# 學生成績管理系統 (Student Grade Management System)

基於C++與STL實作的學生成績管理系統。本專案整合了自訂資料結構、動態陣列容器、標準排序演算法與泛型程式設計，提供直覺的文字輸出介面進行成績統計。

## 功能

1. **新增學生資料 (Add student)**
   - 支援動態輸入學號、姓名與成績。
   - **內建防呆檢查**：若學號重複將拒絕新增，確保資料唯一性。
2. **列出所有學生 (List all students)**
   - 清楚易讀的直式表格排版。
   - 若系統目前無資料，會提示並安全返回主選單。
3. **依成績排序 (Sort by score)**
   - 支援雙向排序調校：**1. 成績由高到低** 或 **2. 成績由低到高**。
   - 採用Lambda匿名函式技術，自訂結構排序。
4. **以學號查詢 (Search by id)**
   - 輸入學號即可快速精準搜尋特定學生的姓名與成績。
   - 若查無資料，系統將輸出錯誤提示。
5. **成績報表統計 (Show statistics)**
   - 自動全盤計算：**全班平均分數**、**及格/不及格總人數**。
   - **Template整合**：計算最高分與最低分統計，呼叫Template函式模板。

## 使用技術與語法

- **自訂資料結構 (`struct`)**：定義 `Student` 結構體封裝學號 (`id`)、姓名 (`name`) 與成績 (`score`)。
- **動態陣列容器 (`vector`)**：使用彈性且記憶體連續的 `vector<Student>` 作為核心資料庫。
- **標準排序演算法 (`sort`)**：結合**Lambda 匿名函式**，靈活切換 `>` 與 `<` 比較運算子，作為雙向排序。
- **泛型程式設計 (`Template`)**：獨立實作 `getMax<T>` 與 `getMin<T>` 函式模板。
- **自動推導變數型別 (`auto`)**：全面採用 `for (const auto& s : students)` 語法，確保**記憶體複製的效能**與**唯讀資料安全性**。
